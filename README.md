# CRUD - PHP
Projeto desenvolvido em PHP que realiza um CRUD de veículos, contendo.
- Número de Chassi 
- Marca
- Modelo
- Ano
- Placa
    - Caracteristicas (sistema exige 2 ou mais caracteristicas para seguir com cadastro)
